Google Drive Nasıl Kurulur?
1
Google Cloud Console'a gidin
console.cloud.google.com → Yeni proje oluşturun (ör: mis-app)
2
Drive API'yi etkinleştirin
APIs & Services → Library → Google Drive API aratın → Enable
3
Servis hesabı oluşturun
APIs & Services → Credentials → + Create Credentials → Service Account → İsim: mis-drive → Role: Editor → Done
4
JSON key indirin
Oluşturduğunuz servis hesabına tıklayın → Keys → Add Key → Create new key → JSON → Create
Dosya bilgisayarınıza inecek.
5
Drive'da MİS klasörü oluşturun
drive.google.com → Yeni klasör: MİS → Klasöre sağ tıkla → Share → JSON'daki client_email adresini ekleyin → Yetki: Editor → Send
6
MİS'e ekleyin
Sağdaki "Hesap Ekle" butonuna tıklayın → JSON dosyasını not defteri ile açıp içeriği yapıştırın → Test butonuyla doğrulayın.
